# 🌐 Web App MVP Tasks (AfriPoks Inspired)

## 🎨 UI/UX Design
- [ ] Setup React project with Tailwind CSS.
- [ ] **Layout**: Dark theme, central octagonal/oval poker table (green/dark felt).
- [ ] **Player Slots**: Circular avatars around the table with:
    - [ ] Username and Chip count.
    - [ ] Card placeholders (back of cards for others).
    - [ ] Dealer button icon.
- [ ] **Center Area**:
    - [ ] Pot amount display.
    - [ ] Community cards (5 slots).

## 🛠️ Functionality
- [ ] Socket.io-client integration.
- [ ] **Action Bar**:
    - [ ] Contextual buttons: Fold, Check/Call, Raise.
    - [ ] Bet slider for custom raise amounts.
- [ ] **Real-time Updates**:
    - [ ] Update UI when receiving new table state.
    - [ ] Animate card dealing (simple CSS transitions).
    - [ ] Highlight the active player.
- [ ] **Showdown**: Display winning hand and highlight winning player.
